## 📖 项目简介

**泪心修改器（TearHeart Trainer）** 是一款专为《七日杀》打造的高品质功能性辅助工具，基于 **Unity IMGUI** 实时绘制界面，采用模块化分页架构。整体由 **透视（ESP）**、**自瞄（Aimbot）**、**杂项（内存修改）** 三大核心面板组成，覆盖视觉增强、瞄准辅助和深度内存改写等全方位修改功能。

> 🌐 **官方下载渠道**：所有正版、最新版本均通过官网统一发布：**[teargamestorem.top](http://teargamestorem.top/)**

## ✨ 功能总览

### 🎯 一、ESP 透视面板（VisualsTab）

通过 Unity IMGUI 实时叠加渲染的完整实体高亮系统。

**全局总控**
- **一键全开 ESP** — 一次性启用所有透视、射线、名称、距离、血条、骨骼透视并切换至 3D 方框模式。
- **全局描边模式**：关闭 / 纯色描边 / 材质描边。
- **全局外框类型**：无方框 / 2D 屏幕方框 / 3D 空间方框。

**实体类透视表格**（玩家 · 敌人 · NPC · 动物）

| 列名 | 说明 |
|------|------|
| 总开关 | 启用该类型透视 |
| 透视射线 | 屏幕指向实体的瞄准引线 |
| 名称显示 | 渲染实体名字 |
| 距离显示 | 渲染实体距离 |
| 血条显示 | 渲染实体血量条 |

- **敌人骨骼透视** — 仅作用于敌人的独立骨架渲染开关。
- 表格采用奇偶行底色区分，关闭项整行变暗，状态一目了然。

**物品容器类透视**（共 6 类：物品 · 补给箱 · 载具 · 战利品袋 · 储物容器 · 矿石）
- 勾选总开关后自动同步开启名称 + 距离显示。
- 取消勾选仅关闭总透视，保留用户原有的名字/距离设置。

**玩家 HUD 信息卡片**
- 总开关：一键启用/关闭整套 HUD 信息显示。
- 四行三列信息布局：
  - **第一行**：等级 · 经验进度 · 可用技能点
  - **第二行**：核心体温 · 游戏阶段 · 世界坐标
  - **第三行**：血月倒计时 · 击杀统计 · 死亡次数
  - **第四行**：本次存活时长

### 🎯 二、自瞄面板（AimbotTab）

两套独立自瞄方案，配置参数完全分离。

#### 🔮 鼠标自瞄（显性自瞄）

| 分组 | 配置项 |
|------|--------|
| **触发设置** | 常驻开启 · 自定义快捷键（实时按键监听绑定） |
| **目标设置** | 对敌人生效 · 对其他玩家生效 |
| **瞄准部位** | 头部 / 脖子 / 胸腔 / 骨盆 |
| **行为设置** | 目标锁定 · 仅可视目标 · 最大作用距离 |
| **FOV 视野限制** | 视野总开关 · FOV 大小调节 · 绘制 FOV 边框 |
| **平滑度** | 平滑系数滑块 — 规避反作弊瞬移检测 |

#### 🌑 静默自瞄（子弹追踪 · 内存坐标修改）

> *又称无声自瞄、子弹穿墙自瞄。不移动鼠标，直接修改子弹弹道在内存中的目标定位坐标，隐蔽性极高。*

| 分组 | 配置项 |
|------|--------|
| **目标设置** | 对敌人生效 · 对其他玩家生效 |
| **瞄准部位** | 头部 / 脖子 / 胸腔 / 骨盆 |
| **行为设置** | 目标锁定 · 仅可视目标 · 最大追踪距离 |

每张卡片均配套独立的动态高度计算，根据开关状态自适应展开/折叠。

### 🎯 三、杂项面板（MiscTab）

左右分栏布局，左侧为轻量级辅助，右侧为深度内存修改作弊（带滚动）。

#### 🟦 左栏 · 游戏辅助
- **创意菜单** 解锁
- **调试菜单** 解锁
- **建造作弊**：建造距离扩展（1–100） · 建造间隔加速
- **拾取作弊**：拾取距离扩展（0–50）

#### 🟥 右栏 · 内存修改作弊（滚动视图）

| 模块 | 功能列表 |
|------|----------|
| **生存类** | 无敌模式 · 无限体力 · 不饿肚子 · 不口渴 |
| **伤害防护** | 无跌落伤害 · 停止流血 · 免疫感染 |
| **角色增强** | 移速倍率 · 伤害倍增 · 一击秒杀 |
| **受击防御** | 伤害抵抗 · 完全免伤无敌 |
| **武器强化** | 无后坐力 · 无限弹药 · 消除散布 · 武器永不损耗 · 连射提速 · 模组上限拉满 |
| **特殊变态** | 超级跳跃 · 吸血效果 · 必定暴击 · 全技能满级 · 无敌移速 |

## 🛠 技术架构

- **渲染引擎**：Unity IMGUI（即时模式实时 GUI 绘制）
- **开发语言**：C#
- **架构设计**：分页化模块结构 · 全局配置持久化
- **稳定性**：所有 UI 绘制环节均带异常捕获，防止菜单崩溃
- **数据持久化**：所有设置自动同步至全局配置参数存储
